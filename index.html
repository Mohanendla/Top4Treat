1. Plates Size

To start, I added a width and height for the plates. Since the plates are block elements, each one of them is in its own line.

2. Flexbox

To start, we should add display: flex to the .table element. .c-table { display: flex; }

Notice that the plates are next to each other. Since flexbox is a single layout direction, the flex items are either aligned in rows or columns. The default is rows.

3. Flexbox - Justify Content

The justify-content works on the main axis of the flex container.

To have an equal spacing between the plates across the main axis, I added justify-content: space-between. See the result in the following figure.

4. Flexbox - Align Items/Self

4.1: Align Items

By default, align-items works on the cross axis. I added align-items: center to center the plates vertically.

4.2:Align Self

Let's suppose that we have some custom alignment requests from our Top4. The first and last plates needs to be aligned differntly. For that purpose, we can use align-self on those flex items.

4. Grid - Justify Items

All the plates should be centred horizontally to their grid area, the first row of plates is close to the top side and the second row is close to the bottom side. By using justify-items, we can control how the plates will be aligned to its grid area.
